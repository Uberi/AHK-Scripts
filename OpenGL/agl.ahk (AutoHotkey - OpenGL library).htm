Overview (agl.ahk)

  • aglInit - Initializes opengl32.dll and GLU32.DLL
  • aglFree / aglCleanup - Is opengl32.dll and GLU32.DLL free

  • aglUseGui - Creates a drawing surface (rendering context) on a AutoHotkey Gui
  • aglUseGuiControl - Creates a drawing surface (rendering context) on a AutoHotkey GuiControl
  • aglUseWindow - Creates a drawing surface (rendering context) on a window

  • aglIsMultisampleSupported - Determine whether hardware-accelerated anti-aliasing is available
  • aglUseGuiMultisample - Creates a drawing surface (rendering context) on a AutoHotkey Gui, when using multisampling (antialiasing)
  • aglUseGuiControlMultisample - Creates a drawing surface (rendering context) on a AutoHotkey GuiControl when using multisampling (antialiasing)
  • aglUseWindowMultisample - Creates a drawing surface (rendering context) on a window when using multisampling (antialiasing)

  • aglGetDC - Get the device context of a window
  • aglReleaseDC - Specifies the device context of a window's released

  • aglInitExt - Initializes the OpenGL extensions (extensions)
  • aglIsExt - Checks if an OpenGL extension (extension) is available
  • aglIsWglExt - Finds whether a Windows OpenGL extension (extension) is available

  • aglGetVersion - Get the version number of the OpenGL implementation
  • aglGetInteger - Returns a single integer value (similar glGetIntegerv)
  • aglGetFloat - Get a single 32-bit floating-point number (similar glGetFloatv)
  • aglGetDouble - Get a single 64-bit floating-point number (similar glGetDoublev)
  • aglGetBoolean - Gets a single Booleanwert (similar glGetBooleanv)

  • aglLoadTexImage1D - Loads a one-dimensional image of a file, for use as a texture
  • aglLoadTexImage2D - Loads a two-dimensional image of a file, for use as a texture
  • aglLoadMipmaps1D - Loads a one-dimensional image of a file, for use as mipmaps
  • aglLoadMipmaps2D - Loads a two-dimensional image of a file, for use as mipmaps
  • aglGenTexture - Generates a single texture
  • aglDeleteTexture - Deletes a single texture
  • aglRenderToTexture - Renders the current scene to a texture

  • aglSetLoadWarnings - Sets the stage for warnings when loading models and materials
  • aglLoadModel - Loads a model from a file
  • aglCallModel - Draw a Model
  • aglDeleteModel - Delete a model
  • aglLoadMaterial - Load a material from a file
  • aglCallMaterial - a material used for drawing
  • aglDeleteMaterial - Deletes a material

  • aglBlur - Draws the contents of the soft color buffer

  • aglLoadProgram - Loads shaders from files or strings
  • aglShaderSource - Sets the source code for a shader object
  • aglGetShaderInfoLog - Get the Infolog the driver for a shader object
  • aglGetProgramInfoLog - Get the Infolog the driver for a program object

aglLoadTexImage1D

Downloads a one-dimensional image of a file, for use as a texture
aglLoadTexImage1D(target, filename [, internalformat=GL_RGBA, minfilter=GL_LINEAR, magfilter=GL_LINEAR] )
target: The target texture (or GL_TEXTURE_1D GL_PROXY_TEXTURE_1D).
filename: The path / name of the file to be used.
internalformat: The format in which it is stored internally OpenGL (to GL_RGB, GL_RGBA, GL_ALPHA, GL_LUMINANCE, GL_INTENSITY, ...).
minfilter: The reduction filter for the texture.
magfilter: The magnification filter for the texture.
Return value: Returns the texture number back, or 0 if an error has occurred.
aglLoadTexImage1D lädt eine eindimensionale Grafik von einer Datei, zur Verwendung als Textur. aglLoadTexImage1D charges a one-dimensional image of a file, for use as a texture.
Supported formats include BMP, GIF, JPG, JPEG, PNG, uncompressed TGA, TIF, TIFF, if GDI + is available on the system.
If GDI + is not available, only uncompressed BMP and TGA to load.
If the OpenGL version 1.2 is smaller than the color of BGR to RGB using GDI + to be converted.
If in this case, GDI + is not available, the function fails and returns 0 (no texture) back.
If the height and width of the image is not a multiple of 2 (2, 4, 8, 16, 32, 64, ...), the GL version is less than 2.0, the extension GL_ARB_texture_non_power_of_two not available, or the graphic is larger GL_MAX_TEXTURE_SIZE as the graphics with GDI + on a compatible size is scaled.
If in this case, GDI + is not available, GDI is used.
If the color depth of the image is not 24 or 32 bit, it will be converted to 32 bit using GDI.
GDI + is Windows XP (Service Pack 2) part of the operating system.Auf älteren Windows-Betriebssystemen kann die gdiplus.dll dem Script (in A_WorkingDir) beigelegt werden (siehe Platform SDK Redistributable: GDI+ ) On older Windows operating systems, the gdiplus.dll to the script (in A_WorkingDir) are included (see Platform SDK Redistributable: GDI + )
The texture is created automatically bound to the selected target texture.
If image is greater than 1 pixel, only the top horizontal line with 1 pixel height is used.
Texturen können kann mit glBindTexture und dem selben target gebunden werden, und mit glDeleteTextures oder aglDeleteTextur gelöscht werden. Textures can be bound to the same target and with glBindTexture, and glDeleteTextures or aglDeleteTextur be deleted.
