Obfuscation of Special Script Objects in Autohotkey Gui Controls

This program can replace special uses of objects used in gui controls.

Gosub Labels in Gui Controls that Start with a 'G'

This program can find and replace occurances of labels found in gui controls that start with a 'G' to signify gosub on click:
 
gui, add, text, Gclickhandler, hello world
 
No special steps are necessary by you for this. An invalid replacement could occur if you have 2 labels that are the same except one starts with a g:
 
lob:
glob:

Global Variables in Gui Controls that Start with a 'V'

This obfuscator can replace global variables starting with a 'V' that are found in GUI controls :
 
GUI, add, text, Vmyglobvar, hello world
 
No extra work is necessary by you for this, just do the normal things required for obfuscating global variables. Only global variables found in gui controls will be replaced at this time, static and local variables starting with a 'V' are not supported.

The GUI +label????? Clause

If you are using the GUI +Label???? clause to set a handler for the closing of a window, you will have to turn off normal label obfuscation encoding for that actual label section. You do this by surrounding that section of code with the change defaults command and the restore defaults command:
 
;$OBFUSCATOR: $LABELS_CHANGE_DEFAULTS: -1
 
configwinClose:
configwinEscape:
        ;program statements here
RETURN
 
;$OBFUSCATOR: $LABELS_RESTORE_DEFAULTS:
 
The -1 in the CHANGE DEFAULTS command turns off obfuscation for label objects that follow and the restore defaults command restores the obfuscation defaults to whatever they previously were.
 
You then use the DEFGUICLOSELAB command comment anywhere in your source code and both the label header and the GUI, +Label?????? statement will be obfuscated:
 
;$OBFUSCATOR: $DEFGUICLOSELAB: (list of close labels)
 
The label headers will come out looking like this, with the CLOSE and the ESCAPE still present:
 
f@kfk#fffffffkk#fkk#kffkClose:
f@kfk#fffffffkk#fkk#kffkEscape:
 
The gui statement will come out looking like this:
 
GUI, +Labelf@kfk#fffffffkk#fkk#kffk

Variables Surrounded with %'s in Autohotkey GUI Controls

Variables surrounded with %'s in controls will be replaced, nothing extra by you needs to be done.
 
gui, add, text, %controlwidth%,
(join`s
bla bla bla....%saythis%
)
