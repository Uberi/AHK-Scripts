sizeof

sizeof() is based on AHK_L Objects and supports both, ANSI and UNICODE version, so to use it you will require Lexikos AutoHotkey_L.exe or other versions based on it that supports objects.

nsizeof is used to calculate the size of structures or data types.
Visit sizeof at AutoHotkey forum, any feedback is welcome.

size:= sizeof(Structure_Definition or Structure_Object)

Parameters

Description
size

The size of structure or data type
Structure_Definition

C/C++ syntax or usual definition (must not be multiline) e.g. "Int x,Int y", C/C++ definitions must be multiline.

Return Value

sizeof returns size of structures or data types
